# Kaiji
Web apps for calculating Big 2 score

# Features
1. Support multiple players ( 0 ~ N )
1. Scoring methods
    1. diff by players
    1. winner takes all
1. Realtime PNL
1. Rules options
    1. Double Chao when card >= 10
    1. Triple Chao when card == 13
1. Offline web application

# Under development
1. Charts for statistic
